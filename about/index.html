Kenapa membuat justblog ?⌗

Saya membuat blog ini sebagai media untuk menyimpan semua pengalaman belajar saya. Sekaligus untuk menyebarkan ilmu bagi siapapun yang membutuhkan. Saya berharap saya tidak malas untuk terus menulis di blog ini ditengah-tengah kesibukan saya hehe.

Ini adalah blog dasar yang saya buat dengan menggunakan.

  • go dan hugo, Bahasa pemrograman Go dan mesin SSG Hugo
  • Terminal Theme, Tema yang digunakan pada blog ini
  • Text Editor, Blog dan semua content ditulis dengan text editor vim
  • PrismJS, Untuk membuat syntax highlighting
  • Blog deploy pada Github

Well, Enjoy the show :)
