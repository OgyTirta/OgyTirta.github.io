Pacman Arch

2020-09-01 :: Ogy Tirta
#Linux 

Manajer paket (seperti apt pada Debian) resmi untuk Arch Linux adalah pacman. Pacman menggabungkan format paket biner sederhana dengan sistem build yang mudah digunakan. Tujuan pacman adalah memungkinkan untuk mengelola paket dengan mudah. Repositori paket dan server cermin dapat diramban di situs web resmi Arch. Arch Linux juga menyediakan repositori berbasis komunitas untuk para pengguna Arch yang disebut Arch User Repository (AUR). Ini berisi deskripsi paket yang memungkinkan penggunanya untuk mengkompilasi paket dari sumber dan kemudian memasangnya melalui pacman. AUR dibuat untuk mengatur dan berbagi paket baru dari komunitas dan untuk membantu mempercepat penyertaan paket populer ke dalam repositori komunitas. Tidak ada pengelola paket resmi untuk AUR, namun terdapat alat bantu AUR seperti yay, pacaur, dan aura yang berfungsi untuk mengelola paket AUR dengan mudah. Alat bantu AUR ini sama sekali tidak didukung oleh Arch Linux.
