Kenapa Menggunakan Linux ?

2020-09-02 :: Ogy Tirta
#linux 

Hi there⌗

Baik sesuai judul, kali ini saya akan membahas Kenapa saya Menggunakan Linux ?. Sejak pertama kali merasakan bangku SMK, saya sendiri menggunakan OS terbaru pada notebook saya yaitu Windows 7, OS terbaru pada jaman-nya. Pada saat itu saya belum mengetahu tentang OS ori/bajakan bahkan untuk praktikum di sekolah saja taunya cuma install ulang windows tanpa tahu itu windows ori/bajakan. Saya mengenal linux pada saat praktikum instalasi OS opensource yang pada saat itu linux yang saya tau hanya debian dan turunannya yaitu ubuntu.

Hal yang saya rasakan saat pertama kali mencoba menginstall linux adalah aneh. ya.. aneh, saya pikir OS ini ketinggalan jaman dan tidak se-Powerfull windows. Pembicaraan dikalangan teman juga tidak jauh-jauh dari yang namanya windows, dimana pada saat itu siapapun yang menggunakan OS windows terbaru dialah yang paling keren wkwkwkw. Tapi hal pahit yang saya rasakan ketika menggunakan windows adalah harus rajin-rajin update antivirus dan scan virus tiap hari, karena sudah beberapa kali merasakan windows saya terjangkit virus dan harus install ulang.

Bagi yang belum paham, gambar diatas adalah deep-freeze, yes saya menggunakan aplikasi tersebut dan berhenti menggunakan antivirus, agak sedikit tricky but lebih efektif daripada antivirus apalagi antivirus-nya hasil crack.

Sejenak saya berpikir kenapa orang-orang bahkan instansi menggunakan windows bajakan dan antivirus bajakan, sedangkan mereka menuntut perangkat mereka berjalan aman dan tanpa gangguan, well semoga pembaca mendapat hidayah dan berhenti menggunakan OS bajakan.

Balik ke topik, kenapa menggunakan linux ?⌗

Semua didasari dari yang namanya berhenti menggunakan produk bajakan, bukan sok suci tapi lebih kepada keamanan data pribadi. emang linux aman? Baik akan saya ceritakan sebuah cerita. windows asli ibarat rumah yang bagus dengan pintu yang bagus dimana anda sebagai pemilik mempunyai kuncinya. sedangkan windows bajakan ibarat rumah yang pintu depannya terkunci dan anda tidak memiliki kunci tersebut namun pintu belakannya terbuka sehingga anda, maling, dan rampok juga bisa masuk. Sedangkan linux adalah OS opensource yang dibangun oleh komunitas, ibaratkan anda mempunyai rumah dengan tetangga yang baik dan saling menjaga.

Linux pertama yang saya gunakan adalah keluarga debian yaitu Ubuntu Pada ubuntu saya hanya mencoba melakukan installasi OS dan installasi aplikasi tanpa berani menggunakannya untuk sehari-hari. karena pada saat itu saya masih banyak menggunakan aplikasi pemrograman seperti visual studio yang berjalan di windows.

Linux kedua yang saya gunakan adalah Manjaro Dimulai dari saya mempunyai laptop sendiri dan terbebas dari praktikum yang menggunakan aplikasi windows, saya pun memberanikan diri untuk menggunakan linux secara full untuk kegiatan sehari-hari. Manjaro is powerfull, merupakan turunan dari Arch dengan paket yang rolling-release artinya selama masih dikembangkan saya masih bisa update hanya dengan menggunakan terminal dan pastinya lebih singkat dibanding update windows.

Tetapi setelah lama menggunakan Manjaro saya mulai kangen dengan environment milik ubuntu, dan pada akhirnya saya bermigrasi ke ubuntu lagi. cukup lama saya menggunakan ubuntu untuk mengerjakan tugas-tugas serta skripsi saya. Karena ubuntu merupakan linux yang cukup powerfull untuk pemula dan tidak banyak config sana-sini. bagi anda yang ingin mencoba saya sangan merekomendasikan ubuntu ini. Tapi kekurangan ubuntu yaitu tidak Rolling-Release artinya setiap versi memiliki umur dan maksimal biasanya 5th setelah itu anda akan berhenti mendapat dukungan update. pada akhirnya juga saya meninggalkan ubuntu karena hal itu, disebabkan saya harus menginstall pembaruan beberapa aplikasi.

ARCH, Linux ketiga yang saya gunakan⌗

Arch merupakan linux yang sangat powerfull, dimana moto dari Arch adalah KISS Keep It Simple Stupid. Menginstall Arch adalah menginstall sistem operasi dasar yang hanya berbentuk CLI. untuk dapat menggunakan GUI, diharuskan untuk menginstall Desktop Environment secara mandiri serta paket-paket lain secara mandiri. sehingga kita dapat membangun dari awal environment yang sesuai dengan kebutuhan kita sendiri, hal ini menyebabkan HDD menjadi lebih lega tanpa harus terisi paket-paket yang tidak dibutuhkan.

Saya akan meng-update pengalaman saya dengan Arch, dikarenakan belum genap sebulan saya menggunakan Arch tetapi saya rasa ini adalah Distro yang sangat menjanjikan.

Jika Kalian ingin menggunakan linux, kalian dapat melihat update tentang linux di Distrowatch. Linux adalah OS yang sangat powerfull banyak sekali distro dengan tujuan tertentu seperti kali untuk hacking, ubutu untuk daily use dan masih banyak lagi. Saya sudah membuktikannya dan merasa terbebas dari jeratan microsoft.
